# 나를 소개합니다 페이지 (with 게시판) (INTRODUCE_PAGE_WITH_PHP)

## 프로젝트 기술
* 나를 소개합니다   
    * PHP, CSS, JAVASCRIPT, MySQL for 게시판 데이터
    * 게시판, 게시글 작성, 수정, 삭제, 읽기 가능
    * 게시글 댓글 작성, 댓글에 댓글 작성 가능
    * 게시글 검색
