# ToyProject
추가 개발을 더한 클론 코딩, 개인 프로젝트를 진행합니다.   

클론 코딩은 따라해보면서 개발을 학습합니다. 그리고 개발을 완료한 뒤에 개인적으로 추가적인 개발을 진행합니다.   

개인 프로젝트로 데이터 베이스 활용 서비스, API 활용 서비스 등을 개발해보면서 경험을 키우고 있습니다.
